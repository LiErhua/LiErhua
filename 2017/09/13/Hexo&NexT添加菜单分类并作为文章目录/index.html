Hexo & NexT添加菜单分类并作为文章目录

发表于 2017-09-13 | 分类于 Hexo
生成默认已有的分类

在NexT中，默认的menu有：首页，归档，分类，标签，关于

以标签 tags 为例：

然后：

  1. 在 tags 文件夹的index.md文档中添加 type: tags
  2. 在 NexT 内的 _config 文档的menu中恢复 tags 的使用
生成没有的分类

然后：

  1. 在 essay 文件夹下的 index.md 文档中添加 “type: essay “
  2. 在NexT的config的menu中添加 essay: /categories/essay
  3. 在NexT的languages中修改中文文件，添加essay的中文显示
  4. 在需要放入 essay 菜单的文章的 front matter 中添加 categories: essay
